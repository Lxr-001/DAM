# 数据空间管理页面

## 一、页面定位

**数据空间管理 = 企业级数据资产的 “文件夹 + 权限 + 配额 + 生命周期” 控制台**

核心目标：**看得清、管得住、用得安全**。

## 二、页面整体结构（推荐）

### 1. 顶部导航

- 产品名 / 平台入口
- 当前模块：**数据空间管理**
- 全局搜索：搜空间名 / 创建人 / 标签
- 用户信息 + 设置

### 2. 左侧菜单（固定）

- 数据空间总览
- 我的空间
- 全部空间（管理员视角）
- 空间回收站
- 空间审批 / 申请
- 数据权限管理

## 三、核心区域：数据空间列表页（最重要）

采用 **卡片 + 表格双模式**，用户可切换。

### 1. 筛选区（顶部）

- 空间类型：私有 / 公共 / 项目 / 部门
- 状态：正常 / 已满 / 即将过期
- 创建时间、创建人、标签
- 快速筛选：我管理的 / 我参与的 / 我收藏的

### 2. 操作区

- 新建数据空间
- 批量删除 / 转移 / 授权
- 导入 / 导出空间清单
- 视图切换：卡片 / 列表

### 3. 列表 / 卡片展示字段（必带）

- 空间名称
- 空间 ID
- 空间描述
- 空间类型
- 容量使用：已用 / 总额（进度条）
- 创建人 & 创建时间
- 成员数
- 权限：管理员 / 编辑 / 只读
- 过期时间（如有）
- 操作：详情 / 编辑 / 权限管理 / 删除 / 转移

## 四、详情页：单个数据空间控制台

进入某空间后，分成**四大标签页**：

### 1. 概览

- 容量、文件数、最近更新
- 快速操作：上传、新建文件夹、权限设置

### 2. 文件管理

- 类网盘结构：文件夹 + 文件列表
- 支持预览、下载、移动、删除、版本管理

### 3. 成员与权限

- 添加成员 / 部门 / 角色
- 权限配置：查看 / 编辑 / 管理 / 删除
- 权限继承、批量设置

### 4. 空间设置

- 重命名、修改描述
- 容量扩容 / 申请配额
- 生命周期：自动清理、过期策略
- 安全设置：水印、下载限制、审核

## 五、交互与体验关键点

- **容量预警**：使用率 > 80% 标黄，>95% 标红
- **权限强提示**：无权限按钮置灰并说明原因
- **批量操作**：支持勾选多个空间统一处理
- **面包屑**：保证用户知道自己在哪
- **空状态友好**：无空间时引导 “创建空间”
- **操作日志**：关键行为留痕（谁、何时、做了什么）

## 六、适合直接用的前端风格

- 主色：深蓝 / 科技蓝（数据平台安全感强）
- 布局：**顶部 + 左侧固定 + 右侧内容区滚动**
- 组件：表格、卡片、进度条、弹窗、树形权限、标签筛选
- 风格简洁、白底、轻阴影，符合企业后台规范

### 总结

1. 数据空间管理页面核心是实现数据资产的可视化管理、权限管控和生命周期管理，整体遵循 “看得清、管得住、用得安全” 的目标；
2. 页面结构分为顶部导航、左侧菜单、核心列表区和详情标签页，筛选、操作、展示字段需覆盖空间管理全维度；
3. 交互设计需注重容量预警、权限提示、批量操作等细节，前端风格贴合企业后台简洁、实用的特点。
