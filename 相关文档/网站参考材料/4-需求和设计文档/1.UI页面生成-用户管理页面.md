# 用户管理页面

1. 左侧为树形结构（可折叠/展开），展示组织架构
   
   - 涵盖功能：新增、编辑、删除、搜索
   
   - 选中组织节点会高亮显示
   
   - 包含字段
     
     - 组织编码
     
     - 组织名称

2. 右侧整体为用户列表，展示左侧选中某组织中的全体用户
   
   1. 列表顶部为搜索框和新增按钮
   
   2. 右侧为列表，展示某组织架构中的用户列表，包括以下字段
      
      - 用户名
      
      - 岗位/职务
      
      - 状态—下拉选择（正常、锁定、停用），默认正常
      
      - 角色—支持多选（数据管理员、数据空间管理员、数据Owener、数据管家、数据使用者）
      
      - 安全等级—下拉选择（绝密、机密、内部、公开），默认公开
      
      - 权限生效日期—日历显示
      
      - 权限过期日期—日历显示
   
   3. 同时根据上面的字段绘制新增和编辑页面
      
      - 要求点击新增按钮，右侧滑动弹出页面
      
      - 要求点击编辑按钮，右侧滑动弹出页面
   
   4- 功能还包括：分页功能、列配置功能、删除
