# 数据空间用户页面—设计文档

## 一、页面概述

- **数据空间用户页面**是数据空间管理体系的核心子页面，主要承载系统使用者对指定数据空间内用户的全生命周期管理，包括用户邀请、权限可视化、用户移除等核心操作，旨在保障数据空间的用户权限管控精准、操作流程高效，同时与数据空间管理主页面形成数据闭环，满足企业级数据空间的精细化运营需求。

- **数据空间用户页面**为SpaceUserList.vue，生成位置在（DAM-vue/src/views/DAM/DataInfo/Space/SpaceUser/SpaceUserList.vue）

## 二、页面跳转逻辑

### 2.1 跳转入口

从**数据空间管理页面**触发跳转，操作规则如下：

- 触发方式：**双击**数据空间管理页面中的用户的icon即可跳转到**数据空间用户页面**。
  
  ![](C:\Users\LiXinran\AppData\Roaming\marktext\images\2026-04-15-13-39-32-image.png)

- 跳转参数：携带当前数据空间的**唯一标识（ID）**、**空间名称**、**空间类型（部门 / 项目 / 私有 / 公共）**、**创建人 / 成员信息**、**部门名称**、**部门ID**等核心参数，确保详细页面与原数据空间数据强关联。
  
  | 参数名称    | 说明                |
  | ------- | ----------------- |
  | 数据空间 ID | 数据空间唯一标识          |
  | 空间名称    | 数据空间展示名称          |
  | 空间类型    | 部门 / 项目 / 私有 / 公共 |
  | 创建人 ID  | 数据空间创建人账号 ID      |
  | 创建人姓名   | 数据空间创建人名称         |
  | 成员信息    | 已关联用户 ID 集合       |
  | 部门 ID   | 所属部门唯一标识          |
  | 部门名称    | 所属部门展示名称          |

### 2.2 跳转返回

- 页面左上角配置 “返回” 按钮（icon 为左箭头 +“返回” 文字），点击后返回数据空间管理页面，并保留原页面的筛选、分页状态。
- 浏览器返回按钮适配：支持浏览器原生返回操作，返回后数据空间管理页面状态与跳转前一致。

## 四、功能介绍

### 4.1邀请用户窗口

1. **邀请用户**：主按钮，icon为plus

2. **邀请窗口**为InviteUserModal.vue，生成位置在（DAM-vue/src/views/DAM/DataInfo/Space/SpaceUser/modules/InviteUserModal.vue）

3. 页面可参考DAM-VUE/src/views/system/modules/SelectUserModal.vue页面，只展示还没有被邀请进入空间的用户
   ![](C:\Users\LiXinran\AppData\Roaming\marktext\images\2026-04-15-14-59-27-image.png)

4. 用户已经被邀请进入空间后，则无法重复邀请

5. **一键勾选/取消**功能：
   
   - 点击该按钮前，必须先选择部门，否则提示“请先选择具体部门”
   
   - 首次点击勾选当前部门下所有用户，再次点击取消当前部门下所有勾选；
   
   - 样式：主按钮，icon为√

6. 状态保留：切换部门后，已勾选的其他部门用户勾选状态不清除，仅刷新当前部门列表；

7. 确定按钮：
   
   - 校验勾选用户数量：无勾选时提示 “请选择需要邀请的用户”
   
   - 点击确定后，勾选的用户，则可以邀请进入空间，并在列表展示
   
   - 反馈：邀请成功弹窗提示 “成功邀请 [X] 位用户加入数据空间”，关闭邀请窗口，刷新主表格数据；邀请失败提示 “部分用户邀请失败：[失败原因]”，并展示失败用户列表；

8. 关闭按钮，关闭邀请用户窗口，不保留勾选状态

### 4.2 移除用户

1. **移除用户**：次按钮，icon为delete
2. 点击 “移除用户” 按钮，触发前置校验：表格无勾选用户时，按钮禁用并悬停提示 “请先勾选需要移除的用户”
3. 弹窗配置：居中模态弹窗，标题 “确认移除用户”，内容 “是否将勾选的 [X] 位用户移出「[空间名称]」数据空间？移除后用户将失去该数据空间的访问权限，请谨慎操作！”。
   1. 点击确定后，关闭弹窗，刷新表格（将勾选的用户从表格中移除），同时提示 “已成功移除 [X] 位用户”；移除失败提示 “部分用户移除失败：[失败原因]”，并展示失败用户列表
   2. 点击取消后，则关闭弹窗

### 4.3 一键勾选/取消

1. 点击下，勾选表格中的全部用户（包括跨页用户），再点击一下，取消勾选表格中的全部用户
2. 样式：主按钮，icon为√
