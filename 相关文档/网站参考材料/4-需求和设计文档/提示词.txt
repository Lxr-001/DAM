### 编码约束
- **Entity**：继承 JeecgBoot 基类（如 `JeecgEntity`），使用 `@TableName`、`@TableId` 等 MyBatis-Plus 注解
- **Controller**：继承 `JeecgController<T, S>`（如适用）；使用 `@AutoLog(value = "xxx")` 记录日志；返回值统一用 `Result<?>` 封装

- **文件上传**：使用 JeecgBoot 内置文件上传接口，前端使用 `JUpload` 组件
- **依赖约束**：禁止引入 JeecgBoot 依赖树之外的第三方库

## 达梦 DM8 数据库规范

| 要求 | 说明 |
| --- | --- |
| **建表语法** | 使用 DM SQL 语法；禁止 `AUTO_INCREMENT`、`ENGINE=InnoDB`、反引号包裹 |
| **主键策略** | 使用 `ASSIGN_ID`（JeecgBoot 雪花 ID）或 `IDENTITY`（自增），请根据项目已有表的主键策略选择并说明理由 |
| **数据类型** | 字符串用 `VARCHAR` / `VARCHAR2`，日期用 `TIMESTAMP`，大文本用 `CLOB` |
| **标识符** | 表名/字段名不使用双引号包裹（除非为保留字），避免大小写敏感问题 |
| **中文注释** | 提供 `COMMENT ON TABLE` / `COMMENT ON COLUMN` 语句 |
| **风格探测** | 先查看 DB 配置文件中的数据库连接信息，确认 schema；再查看项目中已有的 SQL 脚本或 Entity 注解，对齐命名风格（如表名前缀） |
